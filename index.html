編成ツールβ

note の戦法タイムライン系ツールを参考にしつつ、GameWith の個別ページと編成ガイドから根拠を取れる範囲だけで組んだ 1部隊用の編成確認ツールです。 秒数や効果時間は公開ソースで確定できなかったため扱わず、主将 / 副将1 / 副将2 / 補佐1 / 補佐2 と前中後列から、 連鎖順、連鎖率、技能条件、重複技能、列依存をまとめて見られるようにしています。

この画面で見るもの

連鎖順 1〜9 のタイムライン、主将と副将の連鎖率、条件を満たす技能 / 満たさない技能、重複技能を同時に見ます。

この版の前提

タイムラインは秒数ではなく、GameWith の「主将 / 副将1 / 副将2」と「連鎖順 早い / 普通 / 遅い」をそのまま順番に落としています。

前列 / 中列 / 後列で有効になる技能と、戦法の追加効果判定に使います。

主将を選ぶと、副将の連鎖率と連鎖順が計算されます。

副将戦法の表示

戦威オン / オフ比較の代わりに、副将戦法を表示対象から外して見比べられます。

連鎖順タイムライン

1〜9 は GameWith の連鎖順ルールです。主将 / 副将1 / 副将2 に置いたときの発動順だけを安全に表示します。
