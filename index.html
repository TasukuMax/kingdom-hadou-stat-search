Kingdom Hadou / SSR + SR Finder

キングダム覇道 SSR / SR ステータス・技能検索

攻撃・防御・戦威・策略の上位2項目で武将を探せる検索アプリです。技能条件にも対応し、 前列・中列・後列、主将・副将・補佐で絞り込めます。魅力は戦力計算に関わらないため除外しています。

総数 SSR SR 対象ステータス: 攻撃 / 防御 / 戦威 / 策略 出典: GameWith 武将一覧 / 技能一覧

検索条件

1つのステータス検索なら 1位一致 / 2位一致、2つのステータス検索なら 完全一致 / 逆順一致 で表示します。 技能条件は同じ画面で同時に絞り込めます。

武将の最も高いステータスに使います。

2つ目の条件を使う場合だけ選んでください。

レアリティ
技能条件

1ステータス検索

最も高いステータスが一致する武将を 1位一致、2番目に高いステータスが一致する武将を 2位一致 として表示します。

2ステータス検索

1位・2位の順番まで一致する武将を 完全一致、上位2項目は同じでも順番が逆の武将を 逆順一致 として表示します。

技能条件フィルタ

前列・中列・後列と、主将・副将・補佐で絞り込みできます。選んだ条件に一致した技能名もカード内に表示します。

データ出典

GameWith の武将一覧 と 技能一覧 を突き合わせて作成しています。武将の所持技能は武将一覧、技能の発動条件は技能一覧を参照しています。
