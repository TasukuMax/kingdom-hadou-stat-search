キングダム覇道 SSR ステータス検索

戦力の伸びを意識して、各武将の「上位2ステータス」で検索するローカルアプリです。 魅力は戦力計算の対象外として除外し、攻撃・防御・戦威・策略の4項目だけで順位を作っています。 1項目なら「1位一致 / 2位一致」、2項目なら「順番まで一致 / 逆順一致」で分けて表示します。

SSR 同値判定: 攻撃 → 防御 → 戦威 → 策略 魅力は検索対象外 出典: GameWith 武将一覧 / 個別ページ

検索条件

第2条件は任意です。例えば「攻撃 → 防御」で検索すると、完全一致にはその順番どおりの武将、 ある程度一致には「防御 → 攻撃」の武将が出ます。

1項目検索

完全一致は「1位がそのステータス」、ある程度一致は「2位がそのステータス」です。

2項目検索

完全一致は「1位→2位」が同順、ある程度一致は「上位2つだが逆順」です。

戦力対象

魅力は戦力が伸びない前提で除外しています。 そのため、魅力が元の1位でも次点のステータスを1位として扱います。

データ出典

GameWith の武将一覧 と各個別ページの「最大能力」を使用しています。
