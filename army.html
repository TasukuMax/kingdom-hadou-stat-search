よくある質問

検索されやすい内容を、ページの目的ごとに短くまとめています。

キングダム覇道の戦力検索では何ができますか？

戦力検索ページでは、武将の上位2ステータス、技能条件、連鎖率をもとに主将候補を絞り込めます。 育てたい武将を探す時や、主将適性を比較したい時に使いやすい構成です。

キングダム覇道のガチャシミュレーターはどんな内容ですか？

ガチャシミュレーターでは、シーズン3英傑登用の10連結果、SSR排出、天井、累計回数を確認できます。 本家寄せの結果画面と「もう10連する」導線を入れているので、結果確認も直感的です。

キングダム覇道のランキングは何が見られますか？

ランキングページでは、サーバー1・2・3の個別ランキングと、3サーバー統合の全体ランキングを見られます。 まずは戦闘力に絞って、上位1〜100人の名前と戦闘力を確認できる構成です。

このサイトは公式サイトですか？

このサイトはキングダム覇道ユーザー向けの非公式攻略ツールサイトです。 ゲーム・画像・名称などの権利は各権利者に帰属し、実際のゲーム内数値や仕様変更が入った場合は追って更新します。

自動編成の前提

現在戦力、最大見込み、20秒火力、40秒継戦、攻城DPS、耐久、妨害維持、解除厚み、陣形適合、世間採用寄りを別軸で持ち、最後に方針ごとの重みで統合しています。

実測CSVがある時は現在戦力を優先し、ない時は育成段階と装備状態を使った推定です。共有や画像出力は今見ている案をそのまま使います。

この画面の採点基準
  • 公式指南書の「部隊編成のポイント / 軍勢編成のポイント」、GameWith の編成・補佐・陣形記事を土台にしています。
  • note の編成ツール記事やプレイヤー記事で共通していた「連鎖率」「技能補完」「列適性」「5部隊の役割分担」を重視しています。
  • 現在は powerCurrent / powerPotential / burst20s / sustain40s / siegeDps / ehp / controlUptime / cleanseCoverage / formationFit / metaPrior を別軸で持ち、最後だけ最適化方針ごとの重みで合成します。
  • 結果は固定解ではなく、公開ソースを横断して作ったヒューリスティックです。手持ち・簡易育成入力は反映しますが、厳密な戦力式は公開されていないため推定を含みます。
実測戦力CSVと公開根拠

標準推定は公開データ中心、CSV実測優先は貼り付けた現在値の戦力をそのまま使って高戦力編成を組みます。

見出しは「武将名, 戦力, 攻撃, 防御, 戦威, 策略, 魅力, レベル, 天賦, 秘伝」を認識します。戦力列が欠けていても、4ステータス + Lv + 秘伝から推定補完します。1体でも反映すると自動編成は手持ち限定で組みます。

このモードで根拠にした公開情報
  • 公式指南書③ の「部隊と軍勢をより強くするポイント」で、部隊/軍勢の違いと編成の考え方を確認しています。
  • GameWith 戦闘力記事 の「戦力は最大兵力と各ステータスから算出」を前提にしています。
  • GameWith 補佐記事 の検証値「主将40% / 副将20% / 補佐10%」を部隊内の反映率として使います。
  • 共有リンクにはCSV本文を入れません。別端末へ実測データごと渡す時は JSON バックアップを使ってください。

まず 1部隊候補を大量に採点し、そのあと重複なしで 5部隊に束ねています。25体未満になる条件では作れません。

軍勢の総評

総合スコア、役割充足、列配分、おすすめ陣形、固定武将の活かし方を先にまとめます。

編成診断

評価仕様に寄せて、役割不足、採用基準、陣形比較、信頼度をまとめて見ます。

5部隊の詳細

各部隊の列、主将 / 副将 / 補佐、採用理由、連鎖率、特徴タグ、弱点を一覧で見ます。

主将候補ランキング

今の条件で主将向きの武将を並べ、相性が良い副将ペアまで先読みします。

副将ペア候補

軸主将に対して噛み合う副将2人組を、連鎖率と支援相性込みで候補化します。

別案の軍勢

同じ条件で組んだ上位候補を並べて、どこを捨てて何を取りにいったかを比較できます。

1枠差し替え提案

今の25体を大きく崩さず、1枠だけ変えて改善しやすい候補を出します。

差し替え候補

今回落選した中で、役割不足や最適化方針の不足軸を埋めやすい予備候補を出します。

なぜ落選したか

有力でも今回は採用されなかった武将について、役割重複や適性不足を説明します。
